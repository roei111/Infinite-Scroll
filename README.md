https://infinite-scroll-photos.herokuapp.com/

## Features

- Search images
- Apply filters (sort, orientation, color)
- Infinite scrolling
- Animations

## Usage

### Install Dependencies (Client & Server)

```
npm install
```

### Run

```
npm start
```

## Build

```
npm run build
```
